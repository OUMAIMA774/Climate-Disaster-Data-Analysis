# Climate Disaster Data Analysis

## Overview

Climate Disaster Data Analysis is a Data Science project focused on exploring, analyzing, and predicting the severity of climate-related disasters worldwide. The project uses Python libraries for data processing, visualization, and machine learning to extract insights from climate disaster data and support data-driven decision making.

## Objectives

- Explore and understand global climate disaster data.
- Perform statistical analysis and data visualization.
- Identify relationships between disaster severity and their impacts.
- Build a predictive model to estimate disaster severity.
- Provide an interactive dashboard for data exploration.

## Dataset

The dataset contains information about climate disasters that occurred between 2020 and 2025, including:

- Disaster type
- Country
- Severity level
- Economic impact
- Infrastructure damage
- Number of victims
- Response time
- Event duration

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/climate-disaster-data-analysis.git
```

Navigate to the project folder:

```bash
cd climate-disaster-data-analysis
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run app/streamlit_app.py
```

## Results

The analysis revealed significant relationships between disaster severity, economic losses, infrastructure damage, and human impact. The predictive model successfully captured major trends and provided useful insights into climate disaster patterns.
